Ex:
MostrarDisponivel - get()
EscreveDisponivel - set()

- o atributo que tem operações de leitura e escrita deve ser definido dessa maneira:
public bool disponivel {get; set;}

- quando um atributo trabalha com operações get e set, ela se torna uma propriedade (property) e deve ser nomeado dessa maneira:
public bool Disponivel {get; set;}

isso:
public int DescricaoResumida
